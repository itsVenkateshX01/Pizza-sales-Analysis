# Pizza Sales Analysis using SQL

This project focuses on analyzing a pizza store's sales data using SQL to extract meaningful insights and help improve business performance.

---

# Project Overview

The dataset (CSV format) contains detailed information about pizza orders, including order details, pricing, quantity, and categories.  
Using SQL queries, I performed data analysis to answer key business questions and identify trends that can help increase sales.

---

# Objectives

- Analyze overall sales performance  
- Identify best-selling pizzas  
- Understand customer ordering patterns  
- Find peak order times  
- Suggest strategies to improve revenue  

---

# Key Analysis Performed

- Total number of orders placed  
- Total revenue generated  
- Most popular pizza size  
- Top-selling pizza types  
- Revenue contribution by each category  
- Peak hours and days of orders  
- Average number of pizzas ordered per day  

---

# Tools & Technologies

- SQL (MySQL)
- CSV Dataset
- Data Analysis using queries (JOIN, GROUP BY, SUM, COUNT)

---

# Key Insights

- Identified top-performing pizzas contributing maximum revenue  
- Found the most preferred pizza sizes among customers  
- Determined peak business hours for better staff planning  
- Analyzed category-wise performance for menu optimization   

---

# Outcome

This project helped in:
- Strengthening SQL query skills  
- Understanding real-world business data  
- Gaining insights into sales optimization strategies
